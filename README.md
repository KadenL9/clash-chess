# Chess x Clash Royale

## Mechanics
* Each turn, players will receive 2 elixir
* The player can make a move by doing 1 of 2 things:
  * Make a regular chess move
  * Buy a chess piece
* To buy a chess piece, the player must have enough elixir to afford it and it can be placed in any of the 2 front rows on the player's side
  * pieces can only be placed on empty squares
  * placement that causes check/checkmate are allowed
  * if there are no open squares on the players side, they cannot buy pieces
* Elixir will be used to spawn chess pieces
* Chess Piece Elixir Costs
  * [Pawn] = 1
  * [Knight] = 3
  * [Bishop] = 3
  * [Rook] = 5
  * [Queen] = 9
